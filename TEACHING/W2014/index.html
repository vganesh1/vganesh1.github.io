Compilers (ECE 351)
Winter 2014

 Teaching Homepage|Vijay Ganesh Homepage

This course is about compiler construction. Topics include

  • Basics (Lexical Analysis and Parsing): What is a Compiler? Stages of Compilation. Lexical Analysis. Symbol Tables. Parsing
  • Formal Languages: Regular Languages, Context-free Grammars,...
  • Different Categories of Programming Languages: Imperative, Declarative,...
  • Type Checking: What is typing? Why is it useful? Different forms of Typing
  • Compiler Optimizations and Code Generation: Static analysis for optimization
  • Runtime Support

S2014

 Teaching Homepage   |    Course Homepage

I will teach this course in 4 modules. Each module consists of 4-6 lectures. The modules are:

  • MODULE 1 (Lectures 1-6):  Basic security concepts such confidentiality, integrity, availability. Different types of attack models. Malware. Control-hijack attacks. Software engineering techniques to prevent, detect and recover from attacks

  • MODULE 2 (Lectures 7-10): Basic cryptography

  • MODULE 3 (Lectures 11-15): Security policies and principles of safe design

  • MODULE 4 (Lectures 16-22): Browser and internet security
